## 内容规范

1. 先讲人话，再给数学定义。
2. 每个关键公式至少解释来源或直觉。
3. 每章补一份“看到题目如何启动”的决策流程。
4. 例题必须写清题意翻译、第一步理由、完整运算与结果检查。
5. 现代补充明确标注，不与考研必考范围混淆。
6. 不使用 `superpower / using-superpowers` 相关 skill 或工作流。
7. 章节优先采用“知识主链 → 逐项解释 → 强弱/分情况比较 → 母公式 → 识别顺序 → 一页记忆版”。
8. 检测题不能只留问题，必须提供可折叠的完整答案与验算。
