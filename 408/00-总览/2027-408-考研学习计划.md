# 2027 408 考研学习计划

> [!important] 日期边界
> 本计划从 **2026-07-13** 起执行。2027 年全国硕士研究生招生考试的正式初试日期与当年 408 考纲，必须等待教育部公告确认；本文使用“距离正式初试若干周”的相对安排，不把推测日期写成事实。

## 总体策略

| 阶段 | 建议周期 | 核心目标 | 达标证据 |
|---|---:|---|---|
| 一轮体系 | 8–10 周 | 覆盖 26 章，建立概念与公式框架 | 每章复习清单完成，章末自测能口述 70% |
| 二轮强化 | 7–9 周 | 攻克算法、计算与综合专题 | 专题题能独立写出步骤，错因可归类 |
| 三轮真题 | 6–8 周 | 形成时间分配和稳定得分策略 | 套卷限时完成，错题二次正确率上升 |
| 考前回收 | 最后 2–3 周 | 查漏补缺，保持手感 | 只回看错题、公式、易错条件和高频模型 |

## 二轮：专题化强化

| 专题 | 关联章节 | 训练输出 |
|---|---|---|
| 复杂度与算法设计 | 数据结构 1–8 | 不变量、伪代码、复杂度、边界条件 |
| 树、图、查找与排序 | 数据结构 5–8 | 手动画过程并解释选择依据 |
| 数据表示与运算 | 计组 2 | 位宽、符号、舍入、溢出判断 |
| Cache、页表与 TLB | 计组 3 + OS 3 | 地址分解、命中路径、有效访问时间 |
| 指令流水线 | 计组 4–5 | 数据通路、冒险、周期与加速比 |
| 调度、PV 与死锁 | OS 2 | 状态图、信号量含义、资源分配图 |
| 文件与磁盘 | OS 4–5 + 计组 7 | 地址定位、空间分配、寻道计算 |
| 滑动窗口与 TCP | 网络 3、5 | 序号空间、窗口变化、时间轴 |
| IP 与路由 | 网络 4 + 数据结构 6 | 子网划分、转发表、最短路 |

## 三轮：真题与套卷

1. 先按章节做真题，记录知识点和错因，不只记录答案。
2. 再按年份整套限时训练，固定选择题与综合题的时间预算。
3. 每套题至少完成两次复盘：当天分析，间隔复做。
4. 对答案存在争议时，以题目条件、正式答案和教材考试口径为准。

## 每日模板

| 时段 | 任务 | 产出 |
|---|---|---|
| 第一段 | 闭卷回忆旧知识 | 一页框架或口述录音 |
| 第二段 | 学习新章节/专题 | 带条件的公式、算法或流程笔记 |
| 第三段 | 练习与订正 | 题目步骤、错因、可迁移结论 |
| 睡前 15 分钟 | 快速回收 | 3 个自测问题与次日首要任务 |

## 每周检查点

- [ ] 本周新学章节均完成章末自测。
- [ ] 至少两次闭卷复述，不以“看懂”替代“能写出”。
- [ ] 至少整理一次错题，并安排间隔复做日期。
- [ ] 计算题单位和中间过程完整。
- [ ] 算法题能解释正确性与复杂度。
- [ ] 根据正式公告更新 [[资料来源与版本说明]]，不依据传闻修改考纲。
