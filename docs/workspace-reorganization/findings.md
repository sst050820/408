## 网络研究发现
- 截至 2026-07-13，检索到的最新正式全国硕士研究生招生管理规定是 2026 年版本。
- 该正式规定将“计算机学科专业基础”列为全国统一命题科目。
- 未检索到正式发布的 2027 年 408 考纲，因此不得把预测变化写成官方结论。
- IEEE 754-2019 在 IEEE Standards Association 页面仍标为 Active Standard，可作为浮点技术更新的版本边界。
- POSIX.1-2024 是 The Open Group 当前公开的标准接口说明，适合解释接口边界，但不替代 408 的内部机制模型。
- RFC 8200、8446、9000、9110、9113、9114 可分别支撑 IPv6、TLS 1.3、QUIC 与现代 HTTP 技术更新。
- RFC 5321 与 RFC 5322 用于区分 SMTP 传输信封和 Internet Message 的首部/正文；RFC 9111 用于现代 HTTP 缓存说明。

## 技术决策
| 决策 | 理由 |
|------|------|
| 教育部/研招网优先用于考试政策 | 属于权威一手来源 |
| IETF、IEEE/ISO、官方项目文档用于技术更新 | 降低二手资料错误和过时风险 |
| OCR 内容只作归纳底稿 | 避免照录、OCR 错字及冗余 |
| 每章区分“408 必考”“理解补充”“技术更新” | 保持考试口径清晰 |
| 无法核实的真题年份、题号和频次不写入 | 防止虚构信息 |
